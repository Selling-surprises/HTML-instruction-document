```
Chrome: Ctrl + Shift + N
Firefox: Ctrl + Shift + P
```

## 🔍 检查应用是否正常

### 打开控制台 (F12)

**✅ 正常情况：**
- 无红色错误
- 可能有 Supabase 警告（正常）
- 可能有 DialogContent 警告（可忽略）

**❌ 异常情况：**
- 有红色错误信息
- 有 404 或 504 错误
- 有模块加载失败

### 检查页面显示

**✅ 应该看到：**
- 顶部标题："智能文档编辑器"
- 编辑器工具栏
- 可编辑区域
- 右下角设置按钮

**❌ 不应该看到：**
- 完全空白
- 只有背景色
- 错误页面

## 🛠️ 如果仍然白屏

### 方案 A：完全清理
```bash
1. 关闭所有浏览器窗口
2. 清除所有浏览器数据
3. 重启浏览器
4. 访问应用
```

### 方案 B：尝试其他浏览器
```
✅ Chrome (推荐)
✅ Firefox
✅ Edge
⚠️ Safari (可能有细微差异)
❌ IE (不支持)
```

### 方案 C：检查网络
```
1. 确认网络连接正常
2. 尝试关闭 VPN/代理
3. 检查防火墙设置
```

## 📱 移动端问题

### 如果移动端白屏：
```
1. 清除移动浏览器缓存
2. 尝试桌面模式
3. 更新浏览器到最新版本
4. 尝试其他移动浏览器
```
