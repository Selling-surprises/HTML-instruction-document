# 需求文档

## 1. 工具概述

### 1.1 工具名称
HTML说明文档制作工具

### 1.2 工具描述
基于 Canvas 的 HTML 说明文档编辑器,提供类似 Windows CHM 文档的侧边栏目录导航功能。支持段落、表格、图片等元素的右键操作,提供与 Microsoft Word 相似的编辑体验。包含段落对话框、表格属性对话框等交互组件。工具栏新增网页下载功能,支持输入网页链接将网页内容转为 Markdown 格式并导入文档编辑器,网页中无法直接获取的图片转为 base64 编码后导入。

## 2. 页面结构与功能说明

### 2.1 整体布局
- 左侧:CHM 风格侧边栏目录导航区域
- 右侧:文档编辑区域
- 顶部:工具栏

### 2.2 CHM 风格侧边栏目录导航

#### 2.2.1 目录结构
- 树形目录展示,支持多级嵌套
- 节点展开/折叠功能
- 节点前显示展开/折叠图标(+ / -)
- 点击节点创建并跳转至对应的新文档
- 子页面支持折叠显示

#### 2.2.2 目录操作
- 右键菜单:
  - 新建子节点
  - 重命名节点
  - 删除节点
  - 上移/下移节点
  - 更改标题颜色(打开颜色设置对话框)
- 拖拽排序:支持节点拖拽调整顺序和层级
- 拖拽调换位置:支持鼠标拖动目录标题,调换目录节点位置
- 搜索功能:在目录中搜索关键词,高亮匹配节点

#### 2.2.3 侧边栏交互
- 默认宽度:200px
- 可调整侧边栏宽度(拖拽分隔线),最小宽度 150px,最大宽度 400px
- 可折叠/展开侧边栏(点击折叠按钮)
- 当前选中节点高亮显示
- 点击节点时,创建对应的新空白文档并在编辑区打开
- 横向滚动条:当目录标题内容宽度超出侧边栏宽度时显示横向滚动条
- 竖向滚动条:当目录节点数量超出侧边栏高度时显示竖向滚动条

#### 2.2.4 样式要求
- 背景色:默认 #2c3e50,支持用户自定义颜色
- 节点文字:默认白色,支持用户自定义颜色
- 选中节点背景:#34495e
- 悬停节点背景:#34495e
- 分隔线:1px solid #1a252f
- 展开/折叠图标:三角形或 +/- 符号
- 字体:Microsoft YaHei, sans-serif
- 字号:12px
- 滚动条样式:与系统默认滚动条保持一致
- 毛玻璃效果:仅在设置中启用毛玻璃选项后显示
- 内容不透明度:侧边栏内容不透明度设置对侧边栏背景和文字均生效

**功能要求:**

- 预览区域:实时显示选中颜色效果
- 应用范围选择:
  - 侧边栏背景颜色设置

**界面要求:**
- 标题:颜色设置

- 预览区域:显示当前选中颜色的示例效果
- 应用范围选项:提供「侧边栏背景颜色」、「应用到当前标题」和「应用到所有标题」选项

- 图片格式对话框:大小、位置、环绕方式、旋转等设置
- 插入符号对话框:符号库浏览和插入

### 2.5 工具栏功能

#### 2.5.1 页面跳转功能
- 顶部跳转按钮:快速返回页面顶部
- 底部跳转按钮:快速滚动至页面底部
- 设置开关:用户可在设置中开启或关闭该功能

#### 2.5.2 导入/导出功能
- 导入 Word 文档:支持打开 .docx 文件,将文档内容加载到编辑器中
- 导出 Word 文档:将编辑器中的文档内容保存为 .docx 文件
- 导出 HTML 文件:将文档导出为 HTML 文件,默认显示第一个目录节点对应的文档页面
- 支持完整的文档格式转换,包括段落格式、表格、图片等元素

#### 2.5.3 网页下载功能

**入口位置**
- 在工具栏中新增「网页下载」按钮,与现有导入/导出功能并列展示

**核心功能**
- 网页链接输入:点击工具栏「网页下载」按钮后,弹出输入框,用户输入目标网页的 URL
- 网页内容抓取:获取目标网页的 HTML 内容
- Markdown 转换:将网页 HTML 内容解析并转换为 Markdown 格式文本
- 图片处理:
  - 优先尝试直接获取网页中的图片资源
  - 若图片因跨域或其他原因无法直接获取,则将图片转为 base64 编码后嵌入 Markdown 内容中
- 导入编辑器:转换完成后,将生成的 Markdown 内容导入文档编辑器,渲染为可编辑的文档内容

**交互流程**
1. 用户点击工具栏「网页下载」按钮
2. 弹出输入对话框,包含 URL 输入框及「确定」/「取消」按钮
3. 用户输入网页链接后点击「确定」
4. 显示加载状态提示(如「正在获取网页内容…」)
5. 抓取并转换完成后,将 Markdown 内容导入编辑器
6. 若抓取或转换失败,显示错误提示信息,用户可重新输入或取消

## 3. 业务规则与逻辑

### 3.1 目录与文档关系规则
- 每个目录节点对应一个独立的文档
- 点击目录节点时,创建对应的新空白文档并在编辑区打开
- 新建目录节点时,自动创建对应的空白文档
- 删除目录节点时,同步删除对应的文档内容
- 重命名目录节点时,同步更新对应文档的标识
- 不同目录节点的文档内容相互独立,互不影响

### 3.2 目录拖拽规则
- 支持鼠标拖动目录标题,调换目录节点位置
- 拖拽过程中显示拖拽指示器,标识目标插入位置
- 拖拽可跨层级调整节点位置和层级关系
- 拖拽至非法位置时显示禁止拖拽图标,松开鼠标后节点恢复原位置
- 拖拽完成后,目录顺序和层级关系立即更新

### 3.3 目录滚动条规则
- 横向滚动条:当目录节点标题内容宽度超出侧边栏当前宽度时自动显示
- 竖向滚动条:当目录节点总高度超出侧边栏可视区域高度时自动显示
- 滚动条仅在需要时显示,不占用额外空间
- 滚动条样式与系统默认样式保持一致

### 3.4 侧边栏背景颜色与文档标题颜色规则
- 侧边栏背景颜色支持自定义设置
- 文档标题支持自定义颜色设置
- 通过右键菜单「更改标题颜色」或设置入口打开颜色设置对话框
- 对话框提供页面内置色块(包含白色色块)、取色器和不选择任何颜色三种选择方式
- 对话框提供「侧边栏背景颜色」、「应用到当前标题」和「应用到所有标题」三个应用范围选项
- 选择「侧边栏背景颜色」时,颜色应用到侧边栏背景
- 选择「应用到当前标题」时,颜色仅应用到当前选中的目录节点标题
- 选择「应用到所有标题」时,颜色统一应用到所有目录节点标题
- 选择白色色块时,颜色设置为白色
- 选择不选择任何颜色时,保持默认颜色或清除已设置的颜色
- 颜色设置后点击确定立即应用
- 颜色设置保存到文档数据中,导出 HTML 时保留颜色样式

### 3.5 侧边栏内容不透明度规则
- 侧边栏内容不透明度设置对侧边栏背景和文字均生效
- 内容不透明度调整后,侧边栏背景色和文字颜色的透明度同步变化
- 内容不透明度设置保存到用户设置中,下次打开工具时保持上次设置状态

### 3.6 HTML 导出规则
- 导出的 HTML 文件默认显示第一个目录节点对应的文档页面
- 子页面在目录中默认处于折叠状态
- 用户可通过点击目录节点展开/折叠子页面
- 导出的 HTML 文件包含完整的目录导航功能和文档内容
- 导出的 HTML 文件保留侧边栏背景颜色和文档标题的自定义颜色样式

### 3.7 菜单样式规则

**菜单外观**
- 蓝白配色风格,与 Microsoft Word 2010+ 一致
- 菜单背景:#ffffff
- 菜单项文字:#333333
- 悬停高亮背景:#e5f3ff
- 禁用状态文字:#a0a0a0
- 边框:1px solid #c0c0c0
- 分隔线:1px solid #e0e0e0
- 菜单圆角:2px
- 字体:Microsoft YaHei, sans-serif
- 字号:12px

**子菜单展开**
- 子菜单位于父菜单右侧
- 子菜单保持与父菜单相同的样式
- 支持多级嵌套

**对话框样式**
- 对话框背景:#f0f0f0
- 标题栏背景:线性渐变(#ffffff 到 #e5e5e5)
- 关闭按钮:右上角 × 号
- 内容区域背景:#ffffff
- 分组标题背景:#f5f5f5
- 输入框边框:1px solid #a0a0a0
- 确定/取消按钮:线性渐变背景(#ffffff 到 #e5e5e5),悬停时蓝色高亮

### 3.8 网页下载功能规则
- URL 格式校验:必须为合法的 http:// 或 https:// 开头的链接
- 图片获取失败时,自动降级为 base64 编码处理,不中断整体导入流程
- 转换完成后的 Markdown 内容直接插入到编辑区当前光标位置

### 3.9 侧边栏毛玻璃效果规则
- 侧边栏默认不显示毛玻璃效果
- 仅在设置中启用毛玻璃选项后,侧边栏才显示毛玻璃效果
- 毛玻璃效果开关状态保存到用户设置中,下次打开工具时保持上次设置状态

## 4. 异常与边界情况

| 场景 | 处理方式 |
|------|----------|
| URL 格式不合法 | 输入框下方显示校验提示,禁止提交 |
| 网页无法访问(超时/404等) | 显示错误提示「网页无法访问,请检查链接后重试」 |
| 图片直接获取失败 | 自动转为 base64 编码嵌入,不中断整体流程 |
| 网页内容过大导致转换缓慢 | 持续显示加载状态,转换完成后再导入 |
| 转换结果为空 | 提示「未能从该网页中提取到有效内容」 |
| 侧边栏目录为空 | 显示提示文字「暂无目录,请新建节点」 |
| 删除目录节点时该节点包含子节点 | 弹出确认对话框,提示「该节点包含子节点,是否一并删除?」 |
| 拖拽目录节点至非法位置 | 显示禁止拖拽图标,松开鼠标后节点恢复原位置 |
| 侧边栏宽度调整超出限制 | 达到最小宽度 150px 或最大宽度 400px 时停止调整 |
| 目录标题过长超出侧边栏宽度 | 自动显示横向滚动条 |
| 目录节点过多超出侧边栏高度 | 自动显示竖向滚动条 |
| 颜色设置对话框未选择颜色直接确定 | 保持原有颜色不变 |
| 选择「应用到所有标题」后部分节点已有自定义颜色 | 统一覆盖为新选择的颜色 |
| 选择白色色块 | 颜色设置为白色 |
| 选择不选择任何颜色 | 保持默认颜色或清除已设置的颜色 |
| 设置中未启用毛玻璃效果 | 侧边栏显示纯色背景,不显示毛玻璃效果 |
| 内容不透明度设置为 0 | 侧边栏背景和文字完全透明 |
| 内容不透明度设置为 100 | 侧边栏背景和文字完全不透明 |

## 5. 验收标准

- 侧边栏目录以 CHM 风格树形结构展示,支持展开/折叠、拖拽排序、右键操作
- 侧边栏默认宽度为 200px,可调整范围为 150px 至 400px
- 点击目录节点可创建并打开对应的新空白文档
- 每个目录节点对应一个独立的文档,文档内容相互独立
- 侧边栏可折叠/展开

- 输入非法 URL 时,输入框显示校验提示,无法提交
- 网页无法访问时,显示明确错误提示
- 导入/导出 Word 文档功能正常,格式转换完整

## 6. 本期不实现功能

- 网页下载功能不支持需要登录鉴权才能访问的网页内容抓取
- 网页下载功能不支持动态渲染(JavaScript 渲染)的单页应用内容完整抓取
- 不支持批量网页链接批量导入
- 不支持将 Markdown 内容导入时的样式主题自定义
- 不支持目录节点的批量操作(如批量删除、批量移动)
- 不支持目录导出为独立的目录文件
- 不支持多文档同时编辑与切换
- 不支持编辑区滚动时侧边栏自动定位并高亮当前可视区域对应的目录节点
