# Word 导入导出功能测试指南

## 快速测试步骤

### 测试 1：导出基本文档

1. **准备测试内容**
   ```
   在编辑器中输入以下内容：
   - 一个标题（使用 H1）
   - 几段文字
   - 一个列表
   - 一个简单的表格
   ```

2. **执行导出**
   - 点击右下角设置按钮
   - 找到"导出"部分
   - 点击"导出 Word (.docx)"
   - 确认文件下载

3. **验证结果**
   - 用 Microsoft Word 或 WPS 打开下载的文件
   - 检查内容是否完整
   - 检查格式是否正确

### 测试 2：导出带图片的文档

1. **插入图片**
   - 在编辑器中插入一张图片
   - 调整图片大小

2. **导出文档**
   - 执行导出操作
   - 下载文件

3. **验证图片**
   - 在 Word 中打开文档
   - 确认图片正常显示
   - 检查图片尺寸是否合理

### 测试 3：导入 Word 文档

1. **准备测试文件**
   - 创建一个简单的 Word 文档
   - 包含标题、段落、列表等基本元素
   - 保存为 .docx 格式

2. **执行导入**
   - 点击设置按钮
   - 找到"导入"部分
   - 点击"导入 Word"
   - 选择准备好的文件

3. **验证结果**
   - 检查内容是否完整导入
   - 检查格式是否保留
   - 查看控制台是否有警告信息

### 测试 4：导入导出循环测试

1. **创建测试文档**
   - 在编辑器中创建一个包含多种元素的文档
   - 包括：标题、段落、列表、表格、链接

2. **导出文档**
   - 导出为 .docx 格式
   - 保存文件

3. **清空编辑器**
   - 删除所有内容

4. **重新导入**
   - 导入刚才导出的文档

5. **对比验证**
   - 检查内容是否一致
   - 检查格式是否保留
   - 记录任何差异

### 测试 5：纸张设置测试

1. **测试 A4 竖向**
   - 设置纸张大小为 A4
   - 设置方向为竖向
   - 导出并在 Word 中检查页面设置

2. **测试 A4 横向**
   - 设置方向为横向
   - 导出并验证

3. **测试其他纸张**
   - 测试 A3、Letter、Legal 等尺寸
   - 验证页面大小是否正确

## 预期结果

### ✅ 成功标准

**导出功能：**
- 文件成功下载
- 文件可以在 Word 中正常打开
- 内容完整无缺失
- 基本格式正确保留
- 图片正常显示
- 表格结构完整

**导入功能：**
- 文件成功上传
- 显示"导入成功"提示
- 内容正确显示在编辑器中
- 基本格式得到保留
- 无严重错误提示

### ⚠️ 已知限制

1. **样式转换**
   - 某些复杂 CSS 样式可能无法完全转换
   - Word 特有的样式（如主题颜色）不会保留

2. **图片处理**
   - 超大图片可能导致文件过大
   - 外部链接图片需要网络连接

3. **表格功能**
   - 复杂的表格合并可能需要调整
   - 表格样式可能有细微差异

## 性能基准

### 小文档（<100KB）
- 导出时间：< 1 秒
- 导入时间：< 2 秒

### 中等文档（100KB-1MB）
- 导出时间：1-3 秒
- 导入时间：2-5 秒

### 大文档（1MB-10MB）
- 导出时间：3-10 秒
- 导入时间：5-15 秒
